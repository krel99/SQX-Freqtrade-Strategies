### EMA Trend + Pullback RSI Scalper

Koncept: Trade with the short-term trend (fast EMA over slow EMA). Enter on pullbacks where RSI briefly goes oversold/overbought against the trend, then reverts.

Indicators (on single timeframe):

- EMA fast (e.g., 20)
- EMA slow (e.g., 50)
- RSI (e.g., 14)

```
timeframe = "5m"

populate_indicators(dataframe):
    dataframe.ema_fast = EMA(close, 20)
    dataframe.ema_slow = EMA(close, 50)
    dataframe.rsi = RSI(close, 14)
    return dataframe

populate_entry_trend(dataframe):

    # LONG ENTRY
    buy_signal = (
        (ema_fast > ema_slow)                         # uptrend
        AND (rsi < 35)                                # pullback
        AND (close > ema_fast)                        # price above fast EMA
    )

    # SHORT ENTRY
    sell_signal = (
        (ema_fast < ema_slow)                         # downtrend
        AND (rsi > 65)                                # pullback up
        AND (close < ema_fast)                        # price below fast EMA
    )

    dataframe.enter_long = buy_signal
    dataframe.enter_short = sell_signal
    return dataframe

populate_exit_trend(dataframe):

    # LONG EXIT
    exit_long = (
        (rsi > 60) OR                                  # mean reversion reached
        (close < ema_fast)                             # loss of momentum
    )

    # SHORT EXIT
    exit_short = (
        (rsi < 40) OR                                  # mean reversion reached
        (close > ema_fast)                             # loss of momentum
    )

    dataframe.exit_long = exit_long
    dataframe.exit_short = exit_short
    return dataframe
```
