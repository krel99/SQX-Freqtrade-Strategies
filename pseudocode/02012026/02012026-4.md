### 5-Min Bollinger Band Mean Reversion

Concept: Fade extremes relative to Bollinger Bands in a ranging market.

Indicators:

Bollinger Bands (20 period, 2 std)
ADX (to avoid strong trends)

```
timeframe = "5m"

populate_indicators(dataframe):
    dataframe.bb_mid, dataframe.bb_upper, dataframe.bb_lower = BBANDS(close, 20, 2)
    dataframe.adx = ADX(high, low, close, 14)
    return dataframe

populate_entry_trend(dataframe):

    # LONG ENTRY (buy low)
    buy_signal = (
        (adx < 20)                                    # avoid strong trend
        AND (close < bb_lower)                        # price pierces lower band
    )

    # SHORT ENTRY (sell high)
    sell_signal = (
        (adx < 20)
        AND (close > bb_upper)
    )

    dataframe.enter_long = buy_signal
    dataframe.enter_short = sell_signal
    return dataframe

populate_exit_trend(dataframe):

    # exit near middle band
    exit_long = (close >= bb_mid)
    exit_short = (close <= bb_mid)

    dataframe.exit_long = exit_long
    dataframe.exit_short = exit_short
    return dataframe
```
