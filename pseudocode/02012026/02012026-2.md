### 1-Min MACD + Parabolic SAR Trend-Follow Scalper

Concept: (From common 1m scalping tutorials) Trade when MACD crosses in trend direction and Parabolic SAR confirms.

Indicators:

MACD (e.g., 12, 26, 9)
Parabolic SAR
Optional: EMA 50 as trend filter

```
timeframe = "1m"

populate_indicators(dataframe):
    dataframe.macd, dataframe.macd_signal, dataframe.macd_hist = MACD(close, 12, 26, 9)
    dataframe.sar = PSAR(step=0.02, max_step=0.2)
    dataframe.ema50 = EMA(close, 50)
    return dataframe

populate_entry_trend(dataframe):

    # LONG ENTRY
    buy_signal = (
        (close > ema50)                                # trend up
        AND (macd crosses above macd_signal)           # bullish momentum shift
        AND (sar < close)                              # PSAR below price
    )

    # SHORT ENTRY
    sell_signal = (
        (close < ema50)                                # trend down
        AND (macd crosses below macd_signal)           # bearish momentum shift
        AND (sar > close)                              # PSAR above price
    )

    dataframe.enter_long = buy_signal
    dataframe.enter_short = sell_signal
    return dataframe

populate_exit_trend(dataframe):

    # LONG EXIT
    exit_long = (
        (macd crosses below macd_signal) OR
        (sar > close)
    )

    # SHORT EXIT
    exit_short = (
        (macd crosses above macd_signal) OR
        (sar < close)
    )

    dataframe.exit_long = exit_long
    dataframe.exit_short = exit_short
    return dataframe
```
