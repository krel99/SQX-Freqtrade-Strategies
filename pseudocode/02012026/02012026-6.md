6. VWAP Reversion Strategy (Intraday Futures)
   Concept: Use session VWAP; fade deviations and revert to VWAP, or trend-follow around VWAP.

(For Freqtrade, emulate VWAP per exchange data within the same timeframe.)

Indicators:

Session VWAP (computed from intraday OHLCV)

Standard deviation of price vs VWAP for bands

```
timeframe = "5m"

populate_indicators(dataframe):
    dataframe.vwap = VWAP(high, low, close, volume)          # custom implementation
    dataframe.deviation = close - vwap
    dataframe.dev_std = STDDEV(deviation, 20)
    dataframe.upper_dev = vwap + 2 * dev_std
    dataframe.lower_dev = vwap - 2 * dev_std
    return dataframe

populate_entry_trend(dataframe):

    # LONG ENTRY: price too far below VWAP
    buy_signal = (
        (close < lower_dev)
    )

    # SHORT ENTRY: price too far above VWAP
    sell_signal = (
        (close > upper_dev)
    )

    dataframe.enter_long = buy_signal
    dataframe.enter_short = sell_signal
    return dataframe

populate_exit_trend(dataframe):

    exit_long = (close >= vwap)          # revert to VWAP
    exit_short = (close <= vwap)

    dataframe.exit_long = exit_long
    dataframe.exit_short = exit_short
    return dataframe

```
